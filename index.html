Övningar GitHub

Github egna övningar

Här kommer några till övningar som ni själva kan göra hemma :

Övning 1

  1. Skapa ett nytt repository
  2. Klona repository till din dator
  3. Gör ändringar
  4. Commit och push tillbaka dessa ändringar till GitHub

Övning 2

  1. Skapa ett nytt repository
  2. Klona repository till din dator
  3. Gör ändringar
  4. Commit och push tillbaka dessa ändringar till GitHub
  5. Radera en fil och pusha ändringen till Github
  6. Revert tillbaka till när du hade filen
  7. Pusha igen

OBS !!! Testa använda alla möjliga alternativ, Github Desktop, Visual Studio Code och Git Bash Terminal.

De här övningar hjälper er att förstå flödet mellan GitHub och era datorer...
